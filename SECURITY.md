# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | ❌ |
| 1.0.x   | ❌ |
| 1.1.x   | ✅ |

## Reporting a Vulnerability
To report a security vulnerability, open a new issue about that vulnerability. A moderator will then mark it with the tag Security-Vulnerability.
